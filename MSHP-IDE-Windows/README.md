# Портативная Python IDE (Windows)

## Готово: полностью оффлайн
В этой папке уже лежит распакованный portable Python в `python/`.
Никаких скачиваний при запуске нет.

## Запуск
- Запусти `МШПаха.Оффлайн.bat`.
- IDE запустится из portable Python и будет запускать скрипты тем же интерпретатором.

## Что есть
- Вкладки (несколько файлов)
- Сохранение / открытие
- Запуск программы (F5)
- Ввод между редактором и консолью (многострочный)
- Enter — отправить, Ctrl+Enter — новая строка
- Каждое введённое сообщение отображается в консоли с префиксом `>`
- Нумерация строк
- Подсветка синтаксиса (без автодополнения)
- Русский интерфейс + эмодзи
- Переключатель "Тёмная тема"
- Встроенная панель turtle справа (если в коде есть `import turtle`)

## Структура (внутри MSHP-IDE-Windows/)
```
MSHP-IDE-Windows/
  app/ide.py
  python/            <-- распакованный portable Python
  .runtime/
  МШПаха.Оффлайн.bat
```

## Примечания
- Файлы сохраняются в UTF-8.
- Несохраненные вкладки при запуске выполняются из `.runtime/run_temp.py`.
- Один процесс за раз (кнопка Stop завершает текущий).
- Turtle работает в том же процессе IDE и не открывает отдельное окно.
