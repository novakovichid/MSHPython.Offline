# Портативная Python IDE (macOS)

## Важно
В этой папке уже лежит portable Python **с Tkinter/Tk** для двух архитектур.

## Быстрый старт
1. Сделай скрипт исполняемым: `chmod +x run_ide.sh`
2. Запусти: `./run_ide.sh`

## Что есть
- Вкладки (несколько файлов)
- Сохранение / открытие
- Запуск программы (F5)
- Ввод между редактором и консолью (многострочный)
- Enter — отправить, Ctrl+Enter — новая строка
- Каждое введённое сообщение отображается в консоли с префиксом `>`
- Нумерация строк
- Подсветка синтаксиса (без автодополнения)
- Русский интерфейс + эмодзи
- Переключатель "Тёмная тема"
- Встроенная панель turtle справа (если в коде есть `import turtle`)

## Структура (внутри MSHP-IDE-macOS/)
```
MSHP-IDE-macOS/
  app/ide.py
  python/
    x86_64/          <-- Intel
    aarch64/         <-- Apple Silicon
  .runtime/
  run_ide.sh
```

## Примечания
- IDE оффлайн и ничего не скачивает без твоей команды.
- Если Tkinter не установлен, GUI не запустится.
- Turtle работает в том же процессе IDE и не открывает отдельное окно.
- Можно удалить ненужную архитектуру (`python/x86_64` или `python/aarch64`) для экономии места.
