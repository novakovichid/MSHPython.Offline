# Changelog

Все изменения указаны по дате внесения в репозиторий.

## 2026-01-26
### Turtle
- **Холст черепашки зафиксирован на 400×400 пиксель** — размер больше не зависит от размера окна приложения.
- Диапазон координат: от -200 до 200 по обоим осям (X и Y).
- Установлено минимальное разрешение окна: **820×700 пикселей** для корректного отображения.
- Холст больше не растягивается и не сжимается при изменении размера окна.

## 2026-01-25
### Файлы и режимы
- Добавлен режим временных файлов: авто‑сейв вкладок, индикатор режима и предупреждение при выходе.
- Добавлены «Сохранить все» (Ctrl+Shift+S) и пакетное сохранение временных модулей вместе с импортированными картинками.
- Временные картинки можно импортировать через кнопку; они добавляются в архив.

### Запуск и импорт
- Импорт модулей без сохранения работает корректно: при запуске подхватываются несохранённые вкладки.
- Исправлен запуск проекта, когда `main.py` сохранён, а новые модули ещё нет.

### UI и визуализация
- **Подсветка текущей строки** кода при пошаговом запуске (желтый для светлой темы, синий для темной).
- Вертикальные разделители в панели инструментов для лучшей организации кнопок.
- Исправлена пропорция холста `Turtle` (круг теперь круглый, а не овальный).
- Улучшена темная тема: исправлены цвета диалогов (импорт хэша, список картинок).

- Кнопка «Переименовать модуль».
- Кнопка построчного запуска + кнопка шага «Далее».

### Turtle

### Переименование

## 2026-01-23
### Документация
- Обновлены `docs/USER_GUIDE.md`, `docs/TECHNICAL.md`, `README.md`.
- Добавлены пояснения про ввод в turtle‑режиме, меню «Полезное», отступы Tab, авто‑загрузку файла в пустой `main.py`.

### Turtle‑режим: ввод и стабильность
- `input()` в turtle‑режиме перенаправлен в GUI‑поле ввода, без `readline`.
- Ввод работает и внутри импортируемых модулей (подмена `builtins.input` и глобального `input`).
- Перед запуском turtle очищается очередь ввода, чтобы не тянуть «старые» значения.

### Turtle‑режим: размеры и координаты
- Координаты экрана синхронизируются с размером встроенной панели.
- Центр экрана — (0, 0), размеры соответствуют реальному `Canvas`.
- Дополнительно закреплены `turtle._getscreen` / `turtle._getcanvas`, чтобы не появлялось внешнее окно.

### Редактор и UX
- Кнопка «Остановить» бледная и неактивна, когда ничего не запущено.
- «Открыть» загружает файл в `main.py`, если он пустой.
- Нумерация модулей берёт минимальный свободный номер (после удаления `module1` создаётся снова `module1`).
- Tab по выделению делает отступ всех строк.
- Добавлено меню «Полезное → Сделать отступ».

### Запуск
- Запускается только `main.py`, текст кнопки и пункта меню отражает это.
- Детект turtle теперь учитывает импорты из других вкладок/файлов проекта.
